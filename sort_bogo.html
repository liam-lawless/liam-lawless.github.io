Bogo Sort

Bogosort has no practical use, and is mainly a novelty sorting algorithm maybe used in education. However, Bogosort is succeeded by perhaps one of the strongest sorting algorithms, Quantum Bogosort, which "generates a random permutation of its input using a quantum source of entropy, checks if the list is sorted, and, if it is not, destroys the universe... the use of this algorithm will result in at least one surviving universe where the input was successfully sorted in O(n) time."

Algorithm

The Bogosort "algorithm" consists of two lines of code, but requires a few helper methods or java.util libraries to successfully implement... for the odd reason that someone might want to use this algorithm.

																				
void bogoSort(int[] array) {
	while (!isSorted(array)) {
		shuffle(array);
	}
}

boolean isSorted(int[] array) {
	for (int i = 0; i < array.length - 1; i++) {
		if (array[i] > array[i+1]) {
			return false;
		}
	}

	return true;
}

void swap(int[] array, int a, int b) {
	int temp = array[a];
	array[a] = array[b];
	array[b] = temp;
}

void shuffle(int[] array) {
	for (int i = 0; i <= array.length - 1; i++) {
		swap(array, i, Math.round(math.random() * i));
	}
}

Big-O Notation

In the best case scenario, Bogo Sort has a time complexity of O(n), where it is given an already sorted list. Bogo Sort has an average time complexity of O(n * n!) because the array to be sorted has n! permutations, where n is the size of the array.
