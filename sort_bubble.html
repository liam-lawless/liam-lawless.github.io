Bubble Sort

Bubble sort is one of the most simple sorting algorithms. The algorithm works by comparing adjacent elements from a set of data, and swapping them if the first element is larger than the second. The algorithm iterates through the data set N times, where N is the size of the data set. As a result, this algorith is highly inefficient with larger data sets. Despite being elementary, and primarily used in teaching introductory classes in computer science, bubble sort has minor advantages: low memory, easy implementation, and a best case complexity of N iterations. However, the disadvantages of bubble sort makes it impractical in nearly all scenarios. Bubble sort is slowed further by large values towards the beginning of the data set which takes more swaps to move it to the back of the data set, and as previously stated the algorithm is increasingly inefficient with larger data sets.

Algorithm

Bubble sort can be implemented iteratively in very few lines of code.

																				
void bubbleSort(int array[]) {
	for (int i = 0; i < array.length - 1; i++) {
		for (int j = 0; j < array.length - i; j++) {
			// check if the first value is greater than the second value
			if (array[j] > array[j + 1]) {
				// store a temporary value
				int temp = array[j];

				// swap the values being compared
				array[j] = array[j + 1];
				array[j + 1] = temp;
			}	
		}
	}
}

Big-O Notation

In the best case scenario, bubble sort's time complexity is O(n), only iterating through the entire data set N - 1 times. However, the best-case scenario relies on the list already being sorted. The time complexity for the worst-case scenario is O(n2) because the inner and outer for loops (seen above) both have to iterate O(n) times.
