Merge Sort

Merge Sort is a divide and conquer algorithm that performs two distinct operations. First, the algorithm recursively breaks the data set down to its atomic level, the "divide" portion. If the portion of the data set that is being divided is empty or only has one element left, the recursion stops. Next, the atomic data is then merged together in a sorted order.

Algorithm

The two-part algorithm can be implemented as seen below. 

										
void merge(int[] array, int l, int m, int r) {

	// split the array in to a left and right sub array
	int subL =  m - l + 1; // size of the left sub array
	int subR = r - m; // size of the right sub array
	int[] arrL = new int[subL];
	int[] arrR = new int[subR];

	// populate the sub arrays
	for (int i = 0; i < subL; i++) {
		arrL[i] = array[l + i];
	}

	for (int j = 0; j < subR; j++) {
		arrR[j] = array[m + j + 1];
	}

	// subarray indices
	int i = 0;
	int j = 0;
	int k = l;

	while (i < subL && j < subR) {
		if (arrL[i] <= arrR[j]) {
			array[k] = arrL[i];
			i++;
		} else {
			array[k] = arrR[j];
			j++;
		}

		k++;
	}

	// Remaining left subarray elements
	while ( i < subL) {
		array[k] = arrL[i];
		i++;
		k++;
	}

	// Remaining right subarray elements
	while (j < subR) {
		array[k] = arrR[j];
		j++;
		k++;
	}
}

void mergeSort(int[] array, int l, int r) {
	if (l < r) {

		// middle point of array
		int m = (l + r) / 2;

		// Break down array into subarrays until at atomic value
		mergeSort(array, l, m);
		mergeSort(array, m + 1, r);

		// put array back together
		merge(array, l, m, r);
	}
}

Big-O Notation

Merge Sort's time complexity is not changed in any case in which the algorithm is applied. Therefore its best and worst case is O(n log(n)), making it a very efficient algorithm. Because the algorithm does not contain any loops, Merge Sort performs linearly. For example, if the input data size is doubled, the sorting time will be doubled. Merge Sort will have the same runtime regardless if the input data is already sorted or unsorted.
